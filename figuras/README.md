## Figuras

Esta carpeta agrupa las figuras principales del TFM para facilitar su revisión:

- comparativa de tiempo y memoria entre algoritmos;
- escalabilidad del algoritmo Batch GCD;
- diagramas auxiliares usados en la memoria;
- logotipos institucionales.

Algunas figuras se mantienen también en la raíz del proyecto porque la memoria LaTeX puede referenciarlas desde esa ubicación.
